# MemoApp (Test v1)

Windows 데스크톱용 로컬 메모 앱입니다.  
왼쪽에서 목록을 탐색/필터링하고, 오른쪽에서 메모 내용을 작성/수정하는 **2-Pane(목록+편집) UX**를 제공합니다.

> 상태: Test Version v1

---

## 미리보기(UI)
- 좌측: 검색, 카테고리/우선순위 필터, 메모 카드 리스트
- 우측: 제목/메타(카테고리·태그·우선순위·즐겨찾기), 본문 편집, 상단 액션(저장/내보내기/삭제), 저장 위치 표시

---

## 주요 기능

### 1) 메모 작성/편집
- 새 메모 생성 버튼 제공(하단 `+ 새 메모`)
- 우측 편집 영역에서 제목/본문 작성 및 수정
- 메모 메타 정보 관리:
  - **카테고리** 선택(드롭다운)
  - **태그** 입력(텍스트)
  - **우선순위** 선택(예: 일반 등)
  - **즐겨찾기(★)** 토글

### 2) 빠른 탐색(검색/필터)
- 상단 검색 입력으로 메모 목록 즉시 탐색
- 카테고리 필터(예: 전체/기본/할 일/아이디어 등)로 범위 축소
- 우선순위 필터로 급한 메모만 집중 관리
- “즐겨찾기” 중심으로 빠르게 재접근 가능(UX 기준 “재방문 비용” 최소화)

### 3) 저장/내보내기/삭제
- 상단 액션 버튼:
  - **저장**: 현재 편집 중 메모를 로컬에 저장
  - **내보내기**: 메모를 외부 파일로 추출(형식은 프로젝트 설정에 따름)
  - **삭제**: 선택 메모 삭제
- 현재 **저장 위치 경로**를 UI 상단에서 확인 가능(운영/백업 UX 안정성 향상)

### 4) 메모 리스트 카드(좌측)
- 카드에 요약 정보 표시:
  - 제목
  - 짧은 미리보기(스니펫)
  - 카테고리 배지(예: 기본/할 일/아이디어)
  - 날짜/시간
  - 우선순위/상태 아이콘(표시되는 경우)

## 설치/실행

### 요구 사항
- Windows 10/11
- (빌드 시) .NET 런타임/SDK 버전은 프로젝트 설정에 따름

### 실행(배포본)
1. 릴리즈(zip/exe) 다운로드
2. 압축 해제
3. 실행 파일 실행

### 로컬 빌드(개발)
1. 레포 클론
2. IDE에서 프로젝트 열기(Visual Studio 권장)
3. `Release` 또는 `Debug`로 빌드/실행

> 스크린샷 기준 실행 경로 예시:  
> `...\bin\Release\net10.0-windows\win-x64\`

## UX 설계 포인트(의도)
- **왼쪽은 탐색, 오른쪽은 편집**으로 역할을 분리해 사용자의 인지부하를 낮춤
- 필터(카테고리/우선순위) + 즐겨찾기로 “자주 쓰는 메모” 재접근 시간을 단축
- 저장 위치를 화면에 노출해 “내 데이터가 어디에 있는지” 불안을 제거
